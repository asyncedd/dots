## 📦 Dependencies:

- Stuff included in the standard EndeavourOS install
- Polybar: `yay -S polybar`
- autotiling: `yay -S autotiling`
- picom: `yay -S picom`
- Wmfocus: `yay -S wmfocus`
- rofi: `yay -S rofi`
- catppuccin's rofi: `https://github.com/catppuccin/rofi/tree/main/basic/.config/rofi`
